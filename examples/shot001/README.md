# examples/shot001

This directory is placeholder fixture data for the tutorial.
The files keep Blender/interchange-oriented extensions so path scanning looks realistic.
Their contents are plain ASCII text, not real binary production assets.
`hero_model.blend` is not a valid Blender file and cannot be opened in Blender.
`hero_preview.glb` is not a valid glTF binary file.
`hero_diffuse.png` is not a valid image file.
Use this directory only for manifest, validation, and dry-run export-plan tests.
